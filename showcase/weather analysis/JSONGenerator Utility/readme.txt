Follow below steps to generate consolidated weather data to visualize the random weather data in showcase demo,
1. To download historic weather data of the city, follow below steps.
     I)	Visit https://www.visualcrossing.com/weather/weather-data-services
    II)	Type location, then select date range and then select desired weather information from “Query options” and select “Daily” weather data . Then start downloading the data.
2. Place the downloaded data under “Data Store” folder in this project.
3. Then call the city name from program.cs.
4. You can get the consolidated data from “JSON Datas” folder.
5. Copy the Json file to the "Raw data" folder(wpf-demos\showcase\weather analysis\DataStore\RawData) of weather analysis project.
